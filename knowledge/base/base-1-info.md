### 大宮（埼玉）

#### ▼ エリア総評
- 未経験スタートにおすすめ。都内より採用ハードル低め、客単価もそこそこ高い
- 経営者層・フリー客も多く、競合が少ない分お客さんをつかみやすい
- 将来的に池袋・上野に出るための橋渡しとしても有効
- 黒服レベルはお店による（ララァは高い）

#### ▼ 主要店舗比較

| 店舗 | ジャンル | 採用基準 | 特徴・向いてる子 |
|------|----------|----------|-----------------|
| **ララァ** | ラウンジ風キャバ（1時店） | ★★★★★（エリアトップ） | 客層・単価トップクラス。ノルマ・罰金なし・ワンピースOK・お酒飲まなくてもOK。経験者有利だが完全新規フリーは少なめ。「ホワイト大手企業のサラリーマン（夜職ver）」なイメージ。個人的最推し |
| **ドラゴンクラブ** | 高級キャバ（1時店） | ★★★★☆ | フリー入客数トップクラス・豪華内装・黒服レベルNo.1級。ガツガツ稼ぎたい子向き。系列にアイ・ゴールドあり |

#### ▼ 大宮エリアの採用ランク序列

```
ララァ ≧ ドラゴンクラブ >> 神楽・ゴールド・レジェ（同水準）
エプシロン（ラウンジ）は全体を通じて最高レベル
```

#### ▼ 目的別おすすめ

- **超緩め最優先** → モダンタイムス
- **緩めだけど客層・単価も欲しい** → ララァ
- **ガツガツ稼ぎたい** → ドラゴンクラブ
- **高級でゆったり** → ルパン
- **アットホーム・新店** → 神楽・ゴールド
- **会員制ラウンジで働きたい** → エプシロン

#### ▼ ララァ 時給スライドのコツ

- **本指名・同伴をガンガン取る** → 本指1pt、同伴1pt（実質2pt）が最効率
- **延長を伸ばす** → 延長1pt（60分置き）。飲まなくてもOKなので話上手な子に有利
- **場内指名・チケット返し** → 場内0.5ptだが、チケット返しでバック5,000円
- **出勤日数キープ** → 週4〜5でポイント必要数が下がる傾向
- **イベント・バースデー** → バースデー小計30%バックなどポイント倍増日を狙う
- **スタッフと仲良くなる** → 付け回し優先度上がり、調整相談もしやすくなる

### 池袋

#### ▼ エリア総評
- 東口一時店は未経験でも時給5,000円〜当たり前
- フリー数が非常に多く、お客さんをつかみやすい
- 「西口は客層が悪い」は**誤解** → 正確には「西口に客層の悪いキャバが多い」（制服・いちゃ系が集中している）。お店をちゃんと選べば問題なし

#### ▼ 一時店比較（プリンス・インソウ・その他）

**◆ プリンス系（レッドシューズ・フェアリー・シーサイド）**
- 首都圏最強の集客・フリーダントツ。未経験でも馴染みやすい大手感
- レッドシューズ：個室多数・シャンパン開きやすい・客単価高め・売上折半可能
- フェアリー：2025年オープン・若めキャストが多くコンカフェ系ビジュアルも採用。バック率は売上によって変動
- シーサイド：老舗の高級キャバ・落ち着きややあり・内勤の質が非常に高い・今が最も採用しやすい

```
フリー客入り：プリンス > インソウ
平均客単価：プリンス < インソウ
```

**◆ インソウ系（エンジェルフェザー・ビゼ）**
- ゆったり高級志向の接客が求められる。ビゼはその色が特に濃い
- 落ち着いた客層・接客重視・飲み卓少なめ → 飲みが苦手で接客うまい子に最適
- エンジェルフェザー・ビゼは時給10,000円も出せる

**◆ 不死鳥（旧フェニックス、西口）**
- 客層が池袋で圧倒的に穏やか。平均客単価6〜7万円
- 月収100万プレイヤー数が池袋No.1 → 教育体制が群を抜いている
- 時給ポイントスライド式。現在西口に移転、箱が狭くなった点がデメリット
- ショーは強制ではなく上位キャストの任意参加

**◆ グランデ**
- 系列なし・カジュアル寄りの1時店。若いキャストが多く集客も強め
- プリンス・インソウより採用緩め。バック率のインセンティブも悪くない
- 埋もれにくいメリットあり。未経験にはアリ

#### ▼ 西口 深夜店比較

| 店舗 | 特徴 | 向いてる子 |
|------|------|-----------|
| **グランクラス東京** | ハマー+フェリス統合でリニューアルオープン。豪華内装・採用比較的優しめ・客層西口で単価高め。黒服対応良好。接待・経営者層中心 | 未経験でもOK（4割以上在籍） |
| **ストーム（旧アルドーレ）** | 輩完全NG。週末フリー集客エリア内トップクラス。1時上がり・終電上がり柔軟。黒服対応良好で定着率高い | 「客が怖い・うるさいのは無理」という子 |
| **ノックス** | 時給提示強め・深夜まで稼働・終電上がり「毎回じゃなければ」OK・1時以降送りあり。客層は西口らしい賑やかめ | 稼ぎ重視、フリーで回したい子 |
| **プリームス** | 平均時給4,000円前後・保証しっかり・昼職・学校とのWワーク民多数。永久保証あり | ストレス少なく長く続けたい子 |
| **カナロア** | 客入りそこそこ良い。体入一律3,000円・本採用4,000〜4,500円。黒服が優しくて雰囲気いいお店として有名 | 無難に安定した深夜店を求める子 |

#### ▼ 目的別おすすめ（池袋）

- **未経験・初キャバ** → プリンス系列（シーサイドが今一番採用しやすい）
- **落ち着き系・飲み苦手** → ビゼ or 不死鳥
- **ワイワイ稼ぎたい** → シーサイド・フェアリー
- **西口でゆるく** → ストーム or プリームス
- **西口で稼ぎたい** → ノックス or グランクラス

#### ▼ 採用しやすい順（今現在）

```
シーサイド > フェアリー（今もフェニックス系は変わらず）> フェニックス（採用ランク高め）
```

### 上野

#### ▼ エリア総評
- 高バック率エリア。クラス・グランクリュ・テミスが三強
- 時給10,000円前後 + 通常バック/売上折半の高い方で計算・原価引きなしの店も多い
- 月収100万超えのキャストが結構いる
- 渋谷移籍のステップとして人気。上野で実績を積んで渋谷に出るルートが定番

#### ▼ 主要店舗

| 店舗 | 特徴 |
|------|------|
| **クラス** | 高還元キャバ。人柄重視で未経験でも「熱量」で通る子がいる。ただしルックス要求は高い |
| **グランクリュ** | 質重視で採用は厳しめ。強い |
| **テミス** | 某スカウトの影響で人気上昇中。拡大期なので採用枠も増えており中〜高ランクの経験者には逆に狙い目 |
| **クラッシュ** | ラスベガス風のワイワイ系大箱。蓮より採用緩め。在籍不足でキャスト募集中あり |
| **蓮（錦糸町）** | 錦糸町エリアNo.1集客・店売り。年配客層多め・月収1,000万超え嬢多数。アナウンサー系・お姉さん系がウケる。上野蓮と売上を競うレベル |

### 歌舞伎町（新宿）

#### ▼ エリア総評
- 時給10,000円〜の店舗にフリーを持っていかれがちだが、最近は中堅店もフリー流入あり
- ヴィクシー・セリュックス・美人茶屋・リュアーグあたりは今フリー結構入っている
- お店によって雰囲気が大きく違うので体入複数が必須
- 未経験には難しい（特に新宿の高ランク店以外は客足が悪い）

#### ▼ 主要候補店舗（時給6,000円在籍からの移籍想定）

- **妥当ライン**：美人茶屋・リュアーグ・フォーティーファイブ
- **チャレンジ枠**：ジェントル・アジアン・ディアレスト（容姿・接客力・売上次第）
- **フリー強め**：フォーティーファイブ・茶屋

#### ▼ その他

- **ナナプラ（総武系列）**：区分はキャバだが実態はコンキャバ寄り。時給高めだが引かれものが多い。半強制のアウトドア・外出あり。コンカフェ気分で時給4,000円欲しい子向け。万人にはおすすめしない
- 歌舞伎町キンクイ＝キングダムクイーン（新橋キンクイ＝キングアンドクイーンとは別店舗）

## 8. 働き方・メンタル・その他

### ▼ 細客 vs 太客（どちらを重視するか）

- 精神負荷の強い客は**損切りして良い**
- テイカー（もらうだけの客）は切る。ギバー（くれる客）の常連を大切に
- 人と人のつながりが信頼につながり売上につながる
- 「自分に価値があるから巨額のお金が動く」 → 自信を持つことが大事

### ▼ 寮について

- 空いていると言われても次の日に埋まることもある
- 高ランクのキャストのみ貸し出すパターンも多い
- **個人的な見解**：引越し費用を寮で貯めたらサクッと賃貸契約した方がいい（お店との癒着が強くなるためあまりおすすめしない）

### ▼ 業界全体のトレンド

- ここ2〜3年で夜の業界全体の採用ハードルが上昇
- 容姿基準も即戦力期待も厳しくなっている
- これは女の子側の問題ではなく「店が集客を女の子に依存しすぎている構造」の問題
- ブランクがあっても「最近の売上は？」を聞かれることが多い → 熱量と「これから出勤できる」アピールが重要

### ▼ 未経験者向けの総合アドバイス

1. エリアは大宮・池袋を軸に検討
2. お店は時給だけでなく引かれもの・客層・黒服の質も含めて選ぶ
3. 体入は複数店舗行って肌感を確かめる
4. 一人のスカウトと信頼関係を築いて長期的なサポートを受ける
5. SNSを活用して自分でお客さんを引く力を鍛えると飛躍的に稼げる
