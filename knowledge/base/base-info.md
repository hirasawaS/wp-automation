## 0. ナレッジの使い方・記事生成ルール

### 記事トーン
- 目線：ガルバちゃん（しのつか）が女の子に語りかけるフレンドリーな口調
- 押しつけではなく「あなたに合う場所を一緒に探す」スタンス
- 整形・容姿への言及は「系統が合う・合わない」など前向きな表現に置き換える

### 必須テンプレートフレーズ（記事末に必ず入れる）
```
実際の時給・シフト・引かれものは体入や面接で直接確認してね。
ガルバちゃんは無料で一緒に探してあげるから、まずはLINEで相談してみて！
```

## 5. 給料・バックシステム完全解説

### 用語集（記事でよく使う言葉の定義）

| 用語 | 説明 |
|------|------|
| **時給制** | 出勤時間に応じて固定の時給が支払われる基本形 |
| **売上スライド制** | 売上・ポイントに応じて時給がUP/DOWN。結果が出ないと下がる。 |
| **売上折半制** | 時給 or 売上の一定割合の高い方で計算（プリンス系など） |
| **保証制** | 最初の数ヶ月間は売上に関わらず保証時給が支払われる |
| **永久保証** | 保証期間の縛りがない。売上ゼロでも時給が下がらない。 |
| **日払い** | その日の給料をその日に受け取れる（体入時から対応の店も多い） |
| **週払い** | 週単位で給料を受け取れる（例：毎週土曜払い）  |
| **本指名バック** | 指名客が来店した際に発生するバック（小計の10%など） |
| **場内指名バック** | フリー客をテーブルで指名してもらった際のバック（1,000円など） |
| **同伴バック** | 開店前に一緒に食事・飲みをしてから来店する「同伴」に対するバック（小計の15%など） |
| **ドリンクバック** | お客さんにドリンクを奢ってもらった際のバック |
| **引かれもの** | 給料から差し引かれる経費全般（厚生費・ヘアメイク代・送迎代など） |
| **待機カット** | 席に付かない待機時間を時給から差し引く不当な慣習。良い店はなし。 |

### 実際の移籍例（時給変化）

| ルート | 変化 |

上野と渋谷はラウンジ寄りの接客スタイルが共通で、客層も近い。上野で実績を積んでから渋谷に移籍するルートが定番。渋谷シーサイドは上野からの移籍の受け皿として特に人気。

## 8. 働き方・メンタル・よくある疑問

### Q：飲めないけど働けるの？

大半のキャバクラはフェイクドリンク・ノンアル対応が可能。ただし飲めないと採用基準が上がるエリア（渋谷など）もある。事前確認が大事。飲む量より「雰囲気を作る力」が重要なお店も多い。

### Q：週1〜2でも働ける？

エリア・お店によって大きく異なる。大宮は週3以上が原則の店が多く、週1〜2では採用されにくい。東京エリア（上野グランクリュ・池袋フェアリーなど）や新橋オリーブ・五反田ViViはWワーク・週1〜2にも対応している。

### Q：整形・タトゥーは大丈夫？

- 整形感が強いと渋谷系のお店では合わない場合が多い
- タトゥーは隠せれば対応可の店がある（吉祥寺スペーシアなど）
- 事前に体入前に確認するのがベスト

### Q：寮はある？

空いていると言われても次の日に埋まることもある。高ランクキャストのみ貸し出すパターンも多い。引越し費用を寮で貯めてから賃貸に移行する方が、お店との距離感を保てて長く働きやすい。

### Q：太客と細客、どっちを大事にすればいい？

精神負荷の強い客は損切りOK。テイカー（もらうだけ）を切って、ギバー（くれる）常連を大切に育てる方が長期的な売上につながる。「自分に価値があるから巨額のお金が動く」→ 自信を持つことが大事。

### 業界全体のトレンド（2025〜2026年）

- ここ2〜3年で夜の業界全体の採用ハードルが上昇傾向
- 容姿基準も即戦力期待も厳しくなっている
- 「お店が集客を女の子に依存しすぎている構造」が根本原因
- ブランクがあっても「最近の売上は？」を聞かれることが多い → 熱量と「これから出勤できる」アピールが重要
- SNSで自分でお客さんを引く力を鍛えると収入が飛躍的に伸びる

### 未経験者向け・総合アドバイス

1. **エリアは大宮・池袋を軸に検討する**
2. **お店は時給だけでなく引かれもの・客層・黒服の質も含めて選ぶ**
3. **体入は複数店舗行って肌感を確かめる**
4. **スカウトは1人と信頼関係を築いて長期サポートを受ける**
5. **SNSを活用して自分でお客さんを引く力を鍛える**
