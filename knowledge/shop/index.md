## エリア特性サマリー

| エリア | 特性 | 客単価目安 |
|---|---|---|
| 歌舞伎町 | 深夜・朝キャバも充実。コンカフェ需要高。フリー流入も多い | 2〜10万 |
| 池袋 | 東口中心。30〜50代ビジネス系が多く安定。深夜営業も強い | 3〜8万 |
| 渋谷 | 若年層から経営者まで幅広い。フォーシーズン等の高単価店も | 3〜10万超 |
| 上野 | サラリーマン・30〜50代安定層。1時店が多く働きやすい | 2〜5万 |
| 恵比寿 | 六本木並みの富裕層来店。客単価が都内最高水準 | 8〜10万超 |
| 新橋 | サラリーマン密集地帯。17時開店のガルバが多い | 2〜4万 |
| 秋葉原 | コンカフェの聖地。フリー集客が圧倒的に強い | 1〜4万 |
| 赤坂 | TBS・官庁近く。経営者・TV関係者など落ち着いた富裕層 | 3〜6万 |
| 神田 | 銀座・六本木クラスの富裕層が流入。深夜帯で月100万超え実績店あり | 5万超 |
| 六本木 | 高単価エリア。外国人富裕層も多い | 5万超 |
| 大宮 | 埼玉中心地。1時店でWワークしやすい。競合少なめで稼ぎやすい | 2〜4万 |
| 中野 | 再開発で高単価客増加中。エリア掘り起こし余地あり | 2〜4万 |
