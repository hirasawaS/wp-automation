# 池袋エリア 店舗ナレッジ

## エリア特性

- 東口エリアに店舗が集中（東口徒歩圏内）
- 30〜50代のビジネスマン・不動産・建設・経営者層が多く安定感がある
- 深夜営業が強いエリア（深夜0〜5時の営業店多め）
- プリンスグループとINSOUグループが強い
- 客単価は平均3〜8万円
- 未経験から月収100万超えを達成するキャストが他エリアより多い

**業態**: キャバクラ
**営業時間**: 0:00〜5:00（日曜定休）
**場所**: 池袋

**場所**: 池袋東口

## ビゼ（bisser 池袋）

**業態**: キャバクラ
**営業時間**: 20:00〜1:00（日曜定休）
**場所**: 池袋東口

### 店舗の特徴・コンセプト
- INSOUグループ（美人茶屋系列）
- 2015年7月オープンの大箱高級店
- 池袋エリアトップクラスの客層・客入り・定着率

### お店のいいところ
- エリアトップクラスの客層でお客様の質が高い
- 同伴早め（20:30まで）で5,000円・ボトル小計20%還元と高待遇

| 項目 | 金額 |
|---|---|
| 体験 | 5,000円〜（キャンペーンで7,000円〜） |
| 本入 | 5,000円〜（ポイントスライド制） |
| 保証 | 2ヶ月（延長あり） |
| 同伴バック | 20:30まで5,000円 / 以降3,000円 |
| ボトルバック | 小計20% |
| 送り | 800円〜距離変動・23区内対応 |

### 採用基準
- 20歳以上（19歳誕生日前はNG）
- 採用は辛め・ビジュアルに自信がある方向け
- 綺麗系・清楚系・高身長が特にウケやすい

### 客層
- 接待多め・大人紳士的なサラリーマン中心で高単価

## Club 不死鳥（フシチョウ）

**業態**: キャバクラ
**営業時間**: 19:00〜翌1:00（日曜定休）
**場所**: 池袋

### 店舗の特徴・コンセプト
- 創業22年の高級キャバ「フェニックス」が2026年1月「不死鳥」として完全リニューアル
- 常連・フリーともにエリア上位クラス
- 未経験スタートで月収100万円超えのキャストが池袋で一番多いとされる

### お店のいいところ
- リニューアル期だからこその高時給採用（5,000〜15,000円以上）
- ポイントスライド制で実力が時給に直結
- 常連が多く安定した客入り
- 日払い最大10,000円

### 給料システム
| 項目 | 金額 |
|---|---|
| 採用時給 | 5,000〜15,000円以上 |
| 保証 | 原則2ヶ月 |
| ポイントスライド | 半期獲得ポイント数＝時給（ポイント5→5,000円、ポイント20→20,000円） |
| 日払い | 最大10,000円 |

### 採用基準
- 20〜35歳前後（未成年不可）

### 客層
- 企業社長・経営者・役員クラス・余裕あるサラリーマン層で安定
