# 秋葉原エリア 店舗ナレッジ

## エリア特性

- コンカフェの聖地として全国的に有名
- ポケパラなどのランキングサイトからのフリー集客が圧倒的に強い
- オタク・サブカル系のお客様から一般ビジネスマンまで幅広い
- 客単価は1〜4万円程度
- 秋葉原駅から徒歩圏内に多数の店舗が集中
- 採用基準は高め（ルックス重視）な店舗が多い
- 広範囲送迎対応（千葉・埼玉・神奈川・八王子まで対応する店舗も）

## HuGMo（ハグモ）

**業態**: コンカフェ
**営業時間**: 18:00〜25:00（定休日なし・年末年始あり）
**場所**: 秋葉原

### 店舗の特徴・コンセプト
- 秋葉原エリアNo.1の集客力
- キャバ級時給×コンセプトカフェで稼げる王道ハイブリッド店
- 衣装はメーカー特注のオリジナルオーダーメイド品（無料支給）

### お店のいいところ
- 秋葉原トップクラスの集客力でお客様を作りやすい
- キャバ並みに稼げると評判
- 高品質な衣装が無料支給
- 広範囲送迎（60km圏内）
- レベルによっては家賃補助付き

### 給料システム
| 項目 | 金額 |
|---|---|
| 時給 | 2,000〜4,500円（平均2,500円） |
| 保証 | 2〜3ヶ月 |
| 本指名バック | 小計10%＋1セット毎500円 |
| 場内指名 | 1セット毎100円 |
| ドリンク・ボトル | 高水準（詳細要確認） |
| 送迎 | 60km圏内対応 |
| 家賃補助 | レベルによって付与 |

### 採用基準
- 20代（19歳から可）
- 採用ランクは比較的高め・ルックス重視

### 客層
- サラリーマン中心で落ち着いた雰囲気
- 団体客も多く来るため単価UP狙いやすい
