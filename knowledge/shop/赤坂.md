# 赤坂エリア 店舗ナレッジ

## エリア特性

- TBS・官庁・大企業本社が多い高級ビジネスエリア
- 経営者・会社員・TV関係者など落ち着いた富裕層が中心
- 客単価は3〜6万円程度
- 外立ちなしの店舗が多く、健全な接客スタイルが好まれる
- 赤坂駅・溜池山王駅・赤坂見附駅などアクセス良好
- 夜遅くまで賑わうエリアではないが、しっかり稼げる

## cheer akasaka（チアーアカサカ）

**業態**: ガールズバー
**営業時間**: 20:00〜3:00（日曜定休・振替休日あり）
**場所**: 赤坂

### 店舗の特徴・コンセプト
- 2025年6月OPENの新規ガールズバー
- 女の子の働きやすさを第一に考えたお店作り
- 私服出勤OK・引かれもの無し

### お店のいいところ
- 時給2,500円の永久保証（下がらない安心感）
- 送り無料
- 私服出勤OKでコスト不要
- ノルマ・ペナルティなし
- 週1回3時間〜勤務OK

### 給料システム
| 項目 | 金額 |
|---|---|
| 時給 | 2,500円〜（永久保証・スライドでUPも可） |
| 本指名 | 500円＋オーダー20% |
| 場内指名 | 200円/セット |
| 同伴 | 1,000円 |
| ドリンクバック | 小計20% |
| ボトル・シャンパン | 小計20% |
| 日払い | 3時間3,000円 / 6時間5,000円まで |

### 採用基準
- 18〜30歳程度（高校生不可）

### 補足
- 1:00便と3:00便の2便体制で帰宅も安心

### 客層
- TBSから徒歩圏内でサラリーマン・経営者・TV関係者など幅広い

## canan（カナン）赤坂店

**業態**: ガールズバー
**営業時間**: 19:00〜5:00（年中無休）
**場所**: 赤坂

### 店舗の特徴・コンセプト
- 2013年創業の赤坂老舗ガールズバー
- 広めの箱で客入りが安定
- 可愛いバニー衣装で系統オールOK

### お店のいいところ
- 老舗の知名度×幅広い客層で稼ぎやすい
- 未経験に優しい・ノルマ・罰金一切なし
- 外立ちなし（赤坂なのに！）
- 寮完備（約6万円）で地方からの上京・住み込みに対応
