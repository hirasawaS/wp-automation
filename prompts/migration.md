## 想定読者（ペルソナ）

現在キャバ・ラウンジ等で働いていて、移籍・転店を検討している女性。

- 今のお店に不満がある（時給が低い・客層が合わない・黒服の対応が悪いなど）
- もっと稼ぎたい・環境を変えたいと思っている
- でも「移籍先を間違えたくない」「また失敗したくない」と慎重になっている

- 業界の仕組みはある程度わかっている

**この子が一番恐れていること：**
移籍先を間違える・前の店より条件が悪くなる・スカウトに騙される

## 必ず含める要素（順番通りに構成すること）

### ① 導入：読者の「今の状況」を言語化する

- 移籍を考えている人が持っているリアルな不満・悩みを言語化してから始める
- 「移籍=逃げじゃない。正しく動けば確実に状況は変えられる」という前提を作る
- **絶対にやらない：**「本記事では〜解説します」「移籍を検討されている方へ」系の定型導入

### ② エリア・店舗の実態情報（ナレッジDBから）

以下をこのセクションに自然に盛り込む：

- **エリアの特性**（客層・時給相場・採用しやすい時期など）
- **具体的な店舗名＋特徴＋向いている子のタイプ**
  → 内部リンク用アンカーは `[LINK:店舗名]` 形式で1〜2箇所挿入
- **時給交渉で使えるポイント**（売上証明の活用・採用タイミングなど）

書き方のコツ：「〇〇エリアで言うと〜」「たとえば△△という店は〜」のように、
体験談・観察者の視点で書くと現場感が出る。

### ③ 移籍で失敗しないためのチェックポイント（必須）

読者が「次は失敗したくない」と思っている前提で書く：

- **時給だけで選ぶリスク**：引かれもの・待機カット・客層が合わないと結局稼げない
- **系統・雰囲気の相性が最重要**：合わない店に入ると時給が下がる・指名がつかない
- **スカウトの複数依頼（オークション）のデメリット**：目先の時給が上がっても適正超えで後から調整が入るリスク
- **採用の狙い目タイミング**（ナレッジDBに記載の時期・店舗の拡大期など）

### ④ 移籍の進め方・ステップ（必須）

「じゃあ具体的にどうすればいい？」に答える：

- 売上証明の準備と活用法
- 面接・体入で何を見せれば評価されるか
- 複数店舗を体入するメリット
- スカウトを通す場合の動き方

### ⑤ CV導線：しの・ガルバちゃんへの信頼構築 + LINE誘導

以下の要素を**2段落で自然に**書く：

**伝えること：**
- スカウト歴7年・都内〜埼玉エリアのネットワークあり
- 時給交渉・面接同行・移籍後のフォローまで対応
- 「相談だけ」でもOK・無理な勧誘なし
- 移籍先で失敗したくないならプロに頼むのが一番

**トーンのイメージ：**
「移籍って、情報戦なんだよね。タイミング・お店の選び方・交渉の仕方、全部わかってないと損する。」
「一人で動くより、現場を知ってる人間を使った方が絶対に早い。」

❌ NG表現：「絶対稼げます！」「今すぐ登録！」「〜させていただきます」
✅ OK表現：「まず話だけ聞かせて」「損はさせないから」「一緒に動こう」

## 文章スタイル 禁止事項

| NG | 理由 |
|----|------|
| 「〜することが重要です」「〜を意識しましょう」 | AI感・上から目線 |
| 定型導入（「本記事では〜」） | 経験者は即離脱する |
| 根拠のない時給・店名・条件 | 信頼を一瞬で失う |
| 箇条書きだけで進む構成 | 深みのない記事に見える |
| 移籍を無条件に煽る表現 | 読者の判断を尊重するトーンに |
